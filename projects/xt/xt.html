ЗАДАЧА: доработка интернет-магазина

Выполнено

  • Разработан модуль для отправки сообщения об ошибке
  • Верстка новых страниц и отдельных блоков
  • Кроссбраузерность
  • Адаптивность

Используемые технологии:

  • HTML5
  • CSS3
  • JS, Canvas
  • jQuery
  • Material Design Lite
  • PHP

Сообщение об ошибке

Модуль сообщения об ошибке позволяет пользователю отправить сообщение об ошибке, которую он обнаружил на сайте, или описать свои пожелания для улучшения работы сайта.

При разработке был использована технология Canvas.

Модуль позволяет:

  • заштриховать конфиденциальную информацию;
  • выделить область, на которую необходимо обратить внимание;
  • использовать "карандаш".
